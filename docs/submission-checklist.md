## 5. Demo 视频录制脚本

建议视频控制在 2-4 分钟：

1. 说明作品选择题目二：AI 语音绘图工具。
2. 打开 `http://localhost:5173`。
3. 点击麦克风，说明无需鼠标键盘进行绘图。
4. 演示基础图形：

```text
画一个蓝色圆形放在中间
在它右边画一个红色三角形
把圆形变大一点
撤销上一步
```

5. 演示一笔一笔画和上下文：

```text
向右画一条直线
接着向下画一条曲线
在末端画一个圆
从刚才的圆右边继续画直线
```

6. 展示执行日志和最近 DSL，说明自然语言会被转成结构化绘图操作。
7. 简述设计取舍：不是文生图，也不是语音贴图，而是可编辑的语音绘图过程。
8. 演示或说明 DeepSeek 复杂口令解析：配置 Key 后可把一句复杂中文步骤拆成 DSL，失败时回退本地规则。
9. 简述后续计划：曲线能力、上下文选择、导出 PNG/SVG。

## 6. 提交前最终动作

1. 录制 Demo 视频并上传到可访问平台。
2. 将 Demo 链接补入 README。
3. 确认 GitHub 仓库在评审时可访问。
4. 确认 `main` 分支可运行。
5. 确认所有 PR 描述完整。
6. 运行：

```bash
npm run check
npm run dev
```

7. 用 Chrome 打开：

```text
http://localhost:5173
```
