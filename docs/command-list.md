# 语音指令能力清单

本文档记录声绘板当前计划支持和已经实现的语音指令能力，用于比赛设计文档、Demo 脚本和后续功能验收。

## 1. 基础绘图指令

| 指令能力 | 示例口令 | 当前状态 |
| --- | --- | --- |
| 创建圆形 | 画一个蓝色圆形放在中间 | 已实现 |
| 创建矩形 | 画一个绿色矩形 | 已实现 |
| 创建三角形 | 画一个红色三角形 | 已实现 |
| 创建线条 | 画一条黑色横线 | 已实现 |
| 创建箭头 | 画一个箭头 | 已实现 |
| 创建文字 | 写上你好 | 已实现 |

## 2. 属性控制指令

| 指令能力 | 示例口令 | 当前状态 |

| 修改颜色 | 把圆形改成黄色 | 已实现 |
| 指定位置 | 放在左上角 | 已实现 |
| 移动对象 | 把刚才那个移到右边 | 已实现 |
| 放大对象 | 把圆形变大一点 | 已实现 |
| 缩小对象 | 把三角形缩小 | 已实现 |

## 6. 下一步优先级

| 优先级 | 能力 | 目标 |
| --- | --- | --- |
| P0 | 更稳定的多步骤拆解 | 让一句话创建多个对象更自然 |
| P0 | JSON Schema 校验 | 防止无效 DSL 进入执行层 |
| P1 | 复杂场景布局 | 自动安排房子、树、太阳、人物的位置 |
