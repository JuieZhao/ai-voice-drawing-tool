## 7. 当前取消的能力

| 能力 | 当前处理 |
| --- | --- |
| 太阳、云、树、房子、花、小女孩等组合模板 | 已从当前可调用能力中撤掉 |
| 小猫、小汽车等固定规划对象 | 已改为提示用户用笔画步骤描述 |

这样做是为了把项目核心从“语音贴图”收回到“语音控制绘图过程”。

## 8. 下一步优先级

| 优先级 | 能力 | 目标 |
| --- | --- | --- |
| P0 | DeepSeek 复杂口令解析 | 让一句话拆成可执行路径动作 |
| P0 | DSL 白名单 / JSON Schema 校验 | 防止无效 DSL 进入执行层 |
| P0 | 路径上下文增强 | 更稳定理解上一笔、末端、左侧、右侧 |
| P1 | 通用绘图规划 | 让 LLM 把更多物体拆成路径步骤 |
| P1 | 语音澄清 | 对目标不明确的命令主动追问 |
| P2 | 导出 PNG / SVG | 增加完整产品感 |
| P2 | 更丰富曲线 | 支持贝塞尔曲线、圆弧和路径闭合 |

## 9. 指令验收脚本

每次修改指令解析器后，至少用以下口令手动验证一次：

```text
画一个蓝色圆形放在中间
向右画一条直线
向右画一个五格长的直线
接着向下画一条曲线
在末端画一个圆
从刚才的圆右边继续画直线
在 B2 画一个黄色圆形
把它移到 C3
隐藏坐标
显示坐标
在它右边画一个红色三角形
把圆形变大一点
把三角形改成黄色
撤销上一步
重做上一步
清空画布
```

海龟式画笔验收：

```text
清空画布
用画笔画一个正方形
用画笔画一个三角形
落笔
向前走一百
向右转九十度
向前走六十
换成红色
向右转九十度
向前走六十
抬笔
回到中心
```

复杂指令验收目标：

```text
向右画一条直线，接着向下画一条曲线，再在末端画一个圆
```

配置 `DEEPSEEK_API_KEY` 并通过 `npm run dev` 启动时，该口令应优先走 DeepSeek 拆解；未配置 Key 或网络不可用时，应回退本地规则并保持页面可运行。
